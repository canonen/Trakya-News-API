TrakyaNews API Documentation

You can see all the API's below this section. You dont need any API key to use them.

Json key values are : "baslik" for title,"resim" for image link,"metin" for article,"tarih" for published date

GET

/{type}

Returns a json list of spesific news type.Remove the brackets to use. Key values are "son-dakika" for breaking news,"ekonomi" for economy, "spor" for sport,
"dunya" for global news,"otomotiv" for automotive news.

GET

/{type}/{int}

Or you can simply return the New that given integer Value. For example, 0 for first new, 1 for second new...

GET

/{type}/{int}/{key_value}

You can return spesific value from given type and new. Key values are (title,image,orj-article,date). For example /ekonomi/0/title.

GET

/{type}/{int}/{algorithm}

You can summarize the article by given algorithm.
algorithm names are "alg-luhn","alg-lexrank","alg-textrank","alg-giso","alg-ortayol","alg-aio","alg-lsa".
